# Visualizing sorting algorithms using react hooks
This project is inspired from a video on codedamn's channel where mehul suggest to build a sorting algorithm visualizer to strengthen the understanding of react hooks.

### About the project
This project mainly uses useeffect to create side effects everytime the array is getting sorted. This is noticed as the bar get sorted to its correct posistion every 0.5 milliseconds.The array is again generated using useeffects using random values.These values are then scaled and represenetd as bars.

I would love to extend this project by adding more sorting algorithms in the near future.
